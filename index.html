Manual de Git

  1. Iniciar el Git en : git init
  2. Git status -> para verificar los archivo que contiene
  3. git add "nombre del archivo" -> se utiliza para agregar el archivo
  4. Git status -> para confirmar si se ejecuto bien el procedimiento
  5. git commit -m "Como quieras llamar el archivo" -> Se utiliza para darle nombre al archivo
  6. Luego se pega el primer enlace de gitHub
  7. git remote -v -> Se utiliza para confirmar el enlace
  8. Luego se pega el segundo enlace
  9. git push -> para finalizar
